Changelog v2.0

Nuevas Características
  • Interfaz web completamente nueva y responsiva
  • Búsqueda mejorada con resultados instantáneos
  • Integración mejorada con reproductores de video
  • Historial de animes vistos con base de datos
  • Soporte para múltiples dispositivos (móvil, tablet, desktop)
Mejoras Técnicas
  • Migración de tkinter a Flask web framework
  • API REST para mejor organización del código
  • Base de datos SQLite para persistencia de datos
  • Carga asíncrona de contenido para mejor rendimiento
  • Compatibilidad mantenida con AnimeFLV API

Características Principales

Búsqueda Avanzada

Encuentra animes rápidamente usando la API de AnimeFLV con resultados instantáneos.

Múltiples Reproductores

Totalmente Responsivo

Funciona perfectamente en computadoras, tablets y teléfonos móviles.
